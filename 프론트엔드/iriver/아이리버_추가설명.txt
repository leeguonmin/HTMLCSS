black 기본 : #111
gray 기본 : #999


/*  헤더영역  */
1. Logo 와 GNB 사이 간격: 40px


1. GNB 메뉴 ( 제품, 컨텐츠, 커뮤니티, 고객지원, 쇼핑물 )
      - 이미지 폰트 사용
      - 메뉴 사이 간격은 구현하지 않아도 됩니다
           
2. 상단 탑링크 메뉴 ( 로그인, 회원가입, 마이페이지, 공지사항 )
      - 전체보기는 이미지 폴더 common 에 저장되어 있습니다
      - font-size : 10px;
      - font-weight: 700;
      - color :  #999;;
      

/*  게시판 영역  */

1. 타이틀 부분 ( 각각의 보드박스의 상단 제목영역 )
       - 공지사항, 파워리뷰, 이벤트
       - font-size : 15px;
       - font-weight : 700;
      
2. 리스트 부분 
     1) 설명 영역은 링크가 걸리도록 구현되어야 합니다
     2) 날짜 영역은 링크가 걸리지 않는 비링크 영역입니다
     3) 각 리스트 앞쪽에 Dot(점) 이미지 구현
     4) 넘치는 텍스트는 ... 으로 구현되어야 합니다
      - font-size : 13px;
      - font-weight: 400;
      - color: #999
      - [출시안내] 아스텔엔컨 <- 텍스트 영역만 color : #333

/* 바텀 리스트 영역 */
 1) 각 이미지 사이에는 세로 점선이 구현되어야 합니다 ( 해당 이미지 메인 폴더 dot_line )

 2) 회사 정보 ( 주소, 대표이사 ) 부분과 아래 COPYRIGHT 영역은 이미지 텍스트를 사용해도 좋습니다
 3) 우측에 패밀리사이트 버튼이 있습니다
